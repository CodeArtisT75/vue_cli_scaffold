# Vue_cli_scaffold

## Project setup
```
npm install
```

### it`s better to install sass-cli  globally run:
```
npm install -g sass
```

### Compiles and hot-reloads for development (+ compile sass files)
```
npm run dev
```

### Compiles and minifies for production (+ compile sass files)
```
npm run create
```
